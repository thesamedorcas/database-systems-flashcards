# Database Systems (H) Flashcards

A personal flashcard app I built to study for my Database Systems (H) exam. Built with plain HTML, CSS, and JavaScript, no dependencies.

## Features

- **Flip cards** - click or tap a card to reveal the answer
- **Unit tabs** - filter cards by topic or browse all at once
- **Progress bar** - tracks how far through the current deck you are
- **Keyboard & swipe navigation** - move between cards with arrow keys or swipe gestures
- **Responsive** - works on desktop and mobile

## Topics Covered

| Unit | Topic |
|------|-------|
| Relational Model | Codd's model, schemas, tuples, instances |
| Constraints & Keys | Superkeys, candidate keys, entity & referential integrity |
| Functional Dependencies | Armstrong's axioms, FD inference, finding candidate keys |
| Normalisation | Anomalies, design guidelines, normal forms |
| SQL Basics | Queries, joins, filtering |
| Advanced SQL & Analytics | Window functions, aggregates, analytics |
| Physical Design & Hashing | Storage, hashing strategies |
| Indexing Part I | Index structures, basics |
| Indexing Part II (B+ Trees) | B+ tree construction and search |
| Query Processing | Processing algorithms, cost models |
| Query Optimisation I | Logical optimisation |
| Query Optimisation II | Physical optimisation, plan selection |

## Usage

Open `index.html` directly in any modern browser, no server required.
